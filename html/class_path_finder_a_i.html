Detailed Description

Cria a inteligencia artificial para caminhar no labirinto ate atingir uma celula. A IA foi modelada usando o seguinte:

Projeto: Maze Versao: 1.0b Autores: Leno Raimundo

  • Tempo para tomar uma decisao, ou seja, o tempo para ir para a proxima celula;
  • Probabilidade de tomar a decisao certa, ou seja, a probabilidade de a proxima celula ser a certa;
  • Capacidade de memoria: quantidade de celulas ja visitadas que a AI consegue lembrar
